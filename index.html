Perguntas

O que costumam perguntar.

Preciso saber programar?

Não. Você instala o VS Code + Claude Code (grátis, tutorial visual incluído), conecta a conta num clique e conversa por comandos. Nada de código, chave de API na mão ou planilha.

A IA vai mexer na conta do meu cliente sozinha?

Nunca sem o seu OK. A regra de ouro é inegociável: a IA propõe com o número que justifica, você aprova, e só então ela executa. Tudo fica registrado no histórico. Pausar e budget são reversíveis.

Por que negativar e criar campanha saem em CSV?

Pelo caminho padrão (Windsor.ai), o conector executa ao vivo só pausar/ativar campanha e budget — então negativar e criar campanha o sistema entrega em CSV pronto pro Google Ads Editor (1 clique). Se você conectar pela API direta do Google, essas ações também rodam ao vivo. Em vez de fingir, a gente é claro sobre cada caminho.

Funciona com quantas contas?

Quantas você quiser. Cada cliente vira uma pasta com metas e histórico próprios. Ideal pra quem toca uma carteira inteira.

Preciso de assinatura do Claude?

Sim — o Claude Code roda sobre a sua conta Claude (plano Pro ou superior, à parte). O Kenji.Ads é o sistema de skills que roda em cima dele.

E o Meta Ads / conteúdo / SEO?

Este produto é focado só em Google Ads, de propósito — mais simples de instalar e usar. Se você quer um sistema que cobre Meta Ads, conteúdo, SEO e gestão de agência também, existe a versão completa: o Kenji.OS.
